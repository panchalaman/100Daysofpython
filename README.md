# 100 Days of Python

A 100-day coding challenge in Python — building projects from beginner to advanced level.

## Challenge Progress

| Days | Topics |
|------|--------|
| 1-15 | Python Basics, Data Types, Control Flow |
| 16-30 | Functions, OOP, Error Handling |
| 31-45 | File I/O, CSV, JSON, APIs |
| 46-60 | Web Scraping, Automation, GUI with Tkinter |
| 61-75 | Flask, Databases, REST APIs |
| 76-90 | Data Science, Pandas, Matplotlib |
| 91-100 | Advanced Projects & Capstone |

## Tech Stack

- Python 3
- Flask
- Tkinter
- Pandas, NumPy, Matplotlib
- BeautifulSoup, Selenium
- SQLite, SQLAlchemy

## Structure

Each day's project is organized in its own directory:

```
├── day01/
├── day02/
├── ...
└── day100/
```

## Getting Started

```bash
git clone https://github.com/panchalaman/100Daysofpython.git
cd 100Daysofpython
pip install -r requirements.txt
```
